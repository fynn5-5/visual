# ⭐ ALLES AUF EINEN BLICK — Treuherz Go-Live Spickzettel

Dein kompletter Startzettel. Alles, was du für den Shopify-Shop brauchst — ohne Suchen.

---

## 🧱 1. Shopify-Grundeinstellungen
- **Shopname:** `Treuherz` (Einstellungen → Allgemein)
- **Währung:** EUR (wichtig, kaum änderbar)
- **Plan:** das 1-€/Monat-Angebot nehmen (sonst Basic ~32 €)
- **Slogan:** „Damit dein Liebling sich wohlfühlt."

## 🎨 2. Design (Theme „Dawn")
- **Logo hochladen:** `logo.png` → Header · `logo-mark.png` → Favicon
- **Farben:**
  - Buttons/Akzent: `#E07856`
  - Hintergrund: `#FAF6F0`
  - Text: `#2B2724`
  - Zweitfarbe: `#7C9A82`
- **Startseiten-Reihenfolge:** Hero → Vorteile (EU-Versand · Sichere Zahlung · 14 Tage Rückgabe) → Bestseller → Marken-Story → Bewertungen → Newsletter

## 💳 3. Zahlungen (Einstellungen → Zahlungen)
- **Shopify Payments** (Karte) + **PayPal** + **Klarna** (Pflicht in DACH!)

## 🚚 4. Versand (Einstellungen → Versand)
- Pauschale **3,95 €**, **Gratis ab 39 €**
- Lieferzeit ehrlich: **3–8 Werktage** (EU-Lager)

## 📦 5. Lieferant
- **DSers** (AliExpress) oder **CJ Dropshipping** installieren
- Filter **„Ships From: EU"** (Germany/Czech/Spain) = schnelle Lieferung
- **Muster bestellen** (Qualität + eigene Fotos/Videos)
